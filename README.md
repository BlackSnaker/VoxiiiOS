# Voxii iOS

## RU
**Voxii iOS** - нативный мессенджер для iPhone/iPad, реализованный на SwiftUI и подключаемый к серверу Voxii.

### Возможности
- Авторизация и регистрация
- Личные сообщения (DM): отправка, редактирование, удаление, ответы, реакции
- Вложения и медиа: загрузка файлов, голосовые сообщения, предпросмотр ссылок
- Новости и уведомления внутри приложения
- Раздел друзей: заявки, принятие/отклонение, список друзей
- Настраиваемый URL сервера
- Аудио/видеозвонки и системная интеграция вызовов (в рамках текущей реализации проекта)

### Запуск
1. Откройте `Vixii.xcodeproj` в Xcode.
2. Выберите схему `Vixii`.
3. Укажите сервер в настройках приложения (по умолчанию: `https://voxii.lenuma.ru`).
4. Запустите на симуляторе или устройстве.

## EN
**Voxii iOS** is a native SwiftUI messenger client for iPhone/iPad that connects to a Voxii backend.

### Features
- Authentication and registration
- Direct messages: send, edit, delete, reply, reactions
- Attachments and media: file upload, voice messages, link previews
- In-app News and Notifications screens
- Friends management: requests, accept/reject, friends list
- Configurable backend server URL
- Audio/video calls and system call integration (as implemented in the current project)

### Run
1. Open `Vixii.xcodeproj` in Xcode.
2. Select the `Vixii` scheme.
3. Set your backend URL in app settings (default: `https://voxii.lenuma.ru`).
4. Run on simulator or device.
